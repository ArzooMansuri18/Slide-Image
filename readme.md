<h1>Sliding Image Gallery</h1>

<p>This project is a smooth <b>Image Slider/Carousal</b> created using <b>HTML, CSS and JavaScript.</b> It allows users to navigate through multiple images using next and previous buttons, providing an elegant slideshow effect.</p>

<h2>Features</h2>

<ul>
    <li>Auto and manual image sliding</li>
    <li>Next/Previous navigation buttons</li>
    <li>Smooth transition animation</li>
    <li>Fully responsive layout</li>
    <li>Pure HTML, CSS, and JavaScript (no external libraries)</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Slider structure and navigation buttons</li>
    <li>CSS -> Layout, animation, and responsiveness</li>
    <li>JavaScript -> Slide control logic and event handling</li>
</ul>

<h2>How It Works</h2>

<ul>
    <li>Multiple images are placed inside a container (.slider)</li>
    <li>JavaScript controls the current slide index</li>
    <li>Clicking Next or Previous changes the Image by </li>
    <li>Focus is on UI layout, not API data</li>
</ul>

<h2>How to Run</h2>

<ul>
    <li>Open index.html in your browser</li>
    <li>Click Next or Previous to slide images</li>
    <li>Optionally, add auto-slide functionality in main.js</li>
</ul>

<h2>Future Improvements</h2>

<ul>
    <li>Add auto-play with pause on hover</li>
    <li>Add indicator dots for each image</li>
    <li>Add swipe gestures for mobile devices</li>
</ul>
